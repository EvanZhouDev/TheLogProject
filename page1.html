What are Logarithms?

The definition of a logarithm is defined as follows in the HMH textbook:

The exponent that a specified base must be raised to in order to get a certain value

You will commonly see logarithms denoted like this:

\(\log_b a\)

The subscript \(b\) is the base, and we read this as "log base b of y"

However, what does this function mean?
Assume that you have a logarithmic expression like this: \(\log_2 8\), and are asked to evaluate it.

According to HMH definition, we have to find the number that the base, 2, needs to be raised to in order to achieve the number 8.

This is equal to three, because \(2^3\) is equal to 8. In other words, \(\log_2 8 = x\)  if and only if  \(2^x = 8\). Now, if we were to abstract this, we get the golden rule of logarithms:

\(\log_b a = x\)  if and only if  \(b^x = a\)

You may also see the logarithm function defined without a base. You can assume that this means that the base is 10. Putting this into practice, \(\log 100 = log_{10} 100\). Logarithms can sometimes be evaluated by hand, but more complex equations may be evaluated by a scientific or graphing calculator.

Logarithm Examples

Evaluate \(x\) for each of these equations

\(\log 10 = x\)

Remember! This is equivalent to the following: \(\log_{10} 10 = x\) What power do we raise 10 to to get 10? 1 of course! Therefore, \(x=1\)

\(\log_{2}16 = x\)

Let's turn this into a format that we are used to: \(2^x = 16\). From this, we can easily see that x is 4.

\(\log_{1 \over 2}16 = x\)

This is a little bit more tricky. Again, turn it into an exponential format: \({1 \over 2}^x = 16\). Now we can see that \(x=-4\), as we can get \({1 \over 2}^4 = {1 \over 16}\), and the negative bit takes the reciprocal, giving us 16.
